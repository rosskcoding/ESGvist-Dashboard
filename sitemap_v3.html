v2 → v3 Changelog
What changed from mockup-v2
✅ Added (15 new screens + 1 modal)
• Tasks — personal task list with SLA indicators
• Merge View — element × standard matrix
• Periods — reporting period management
• Notifications — in-app notification center
• Shared Elements — cross-standard reusable data catalog
• Mapping Matrix — shared element × standard
• Disclosures — disclosure requirements management
• Req. Items — atomic decomposition tree
• Deltas — delta requirements management
• Dictionaries — reference data (units, scopes, etc.)
• Form Config — wizard step configuration
• Projects — reporting project creation & lifecycle
• Readiness Check — pre-export drill-down
• SLA Dashboard — deadline compliance & escalation
• Heatmap — coverage by topic × standard
• Impact Modal — preview when adding standard
🔄 Redesigned (7 screens)
• Overview — SLA, notification bell, visual hierarchy
• Collection — reuse badges, outlier flags, SLA colors
• Validation — reuse transparency, threaded comments
• Evidence — binding status, unbound highlight
• Export — readiness check, blocking issues
• Standards — consolidated from SETUP-0/1/2/3 + CUSTOM-1/2/3/4
• Assignments — SLA column, backup owner, escalation
❌ Removed (15 screens consolidated)
• Onboarding → folded into Overview
• WS-1, WS-2, WS-3, WS-ADM → replaced by role-based Overview + Tasks
• SETUP-0/1/2/3 → consolidated into Standards
• CUSTOM-1/2/3/4 → consolidated into Standards
• QN-1/2/3/4 → consolidated into Wizard-QN
• QL-1/2/3 → consolidated into Wizard-QL
• Roles → folded into Users
• Boundary → folded into Company Settings
🏗 Architecture changes
• CSS class system (no inline styles)
• Unified layout: topbar + sidebar on ALL screens
• Notification bell with unread count
• English UI (Russian in data examples only)
• 32 screens → 29 screens (aligned with full TZ coverage)
