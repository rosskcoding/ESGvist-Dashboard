Interactive Export (Offline Package)

This ZIP contains a self-contained HTML report with search and assets.

How to view:
1) Unzip the package to a folder.
2) Start a local web server from that folder:
   - macOS/Linux: run ./start.sh
   - Windows: run start.bat
3) Open the printed URL in your browser (usually http://localhost:8000/).

Notes:
- A local server is required because browsers block some features when opening HTML via file://
- Search index is in assets/search/
